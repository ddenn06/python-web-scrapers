# Python Web Scraper: Quotes Extractor

A fast and structured Python web scraper built to extract data from websites. This project demonstrates core backend and automation skills.

## 📌 Features
- Sends HTTP requests and handles server responses safely.
- Parses raw HTML efficiently using `BeautifulSoup`.
- Extracts specific data points (Quotes and Authors).
- Clean, commented, and easily scalable code structure.

## 🛠 Tech Stack
- **Language:** Python 3
- **Libraries:** `requests`, `beautifulsoup4`

## 🚀 How it works
The script connects to the target URL (`quotes.toscrape.com`), verifies the connection status, isolates the required HTML containers, and iterates through them to extract and print the text data in a clean format.
